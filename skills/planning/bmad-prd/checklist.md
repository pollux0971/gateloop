# bmad-prd completion checklist

The `prd` stage is `done` only when every item passes under the completion
checker (deterministic structural checks; directives after `::`).

- [ ] Overview section present and non-empty :: section: Overview
- [ ] Functional Requirements section present :: section: Functional Requirements
- [ ] Non-Functional Requirements section present :: section: Non-Functional Requirements
- [ ] Success criteria section present :: section: Success criteria
- [ ] Functional Requirements are itemized and testable (FR- markers) :: contains: FR-
- [ ] Non-Functional Requirements are itemized (NFR- markers) :: contains: NFR-
- [ ] no TBD / TODO / placeholder left :: no-tbd
- [ ] primary users captured :: contains: users
- [ ] scope captured :: contains: scope
