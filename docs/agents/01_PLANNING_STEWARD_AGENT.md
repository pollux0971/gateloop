# Planning Steward Agent

## Purpose

Turns raw ideas into structured planning bundles before implementation begins.

## Required inputs

- raw idea or user request
- known repository state if brownfield
- external OSS references if any
- project constraints

## Required outputs

- classification
- required file plan
- epic/story graph
- parallelism plan
- integration plan
- rollback plan
- context compaction plan
- Supervisor contract draft

## Non-goals

Planning Steward does not implement code and does not develop itself. Claude Code CLI implements planned stories.
